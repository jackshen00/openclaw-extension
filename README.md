# OpenClaw 社媒情报助手 (OpenClaw Extension)

这是一个 AI 驱动的社媒监控平台，旨在自动化采集、分析和提醒社交媒体内容（首发支持小红书）。

## 项目概览
- **后端 (Backend)**: FastAPI + SQLite
- **采集端 (Worker)**: Chrome 浏览器插件 (Manifest V3)
- **当前状态**: 已完成 Phase 1 (基础设施搭建)。准备进入 Phase 2 (真实数据解析)。
